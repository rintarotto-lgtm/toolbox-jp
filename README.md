# 🧰 ツールボックス

**[toolbox-jp.vercel.app](https://toolbox-jp.vercel.app)** — 60種類以上の無料オンラインツール集

## 特徴

- 🌐 ブラウザ完結 — データはサーバーに送信されません
- ⚡ 高速 — 静的サイトで即座にレスポンス
- 📱 レスポンシブ — PC・スマホ・タブレット対応
- 🔒 プライバシー重視 — 入力データはローカル処理のみ

## ツールカテゴリ

| カテゴリ | ツール数 | 例 |
|---------|---------|---|
| 📝 テキスト | 17 | 文字数カウンター、テキスト差分比較、ローマ字変換 |
| 💻 開発 | 32 | JSON整形、Base64変換、正規表現テスター |
| 🎨 デザイン | 7 | CSSグラデーション生成、Flexboxジェネレーター |
| 🔒 セキュリティ | 2 | パスワード生成、ハッシュ生成 |
| 🖼 画像 | 2 | 画像リサイズ、画像Base64変換 |

## 技術スタック

- [Next.js](https://nextjs.org/) (Static Export)

- [Tailwind CSS](https://tailwindcss.com/)
- [Vercel](https://vercel.com/) (Hosting)
